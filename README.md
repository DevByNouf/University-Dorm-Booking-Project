# University Dorm Booking Website

A university project designed to help students find suitable housing during their studies.  
This website serves as a link between dorm owners and students, providing clear prices, features, and a simple booking process.

## Features
- Browse and compare internal and external dorms  
- Clear pricing and dorm features  
- Simple and user-friendly booking system  
- Fully documented project with codes, screenshots, planning, analysis, and evaluation
